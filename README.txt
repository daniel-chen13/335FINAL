Description:
This application allows users to login to their dashboard page that displays their to-do list along with a NASA photo of the day and a URL to a news article.

APIs Used:
1. https://data.nasa.gov/Space-Science/Astronomy-Picture-of-the-Day-API/ez2w-t8ua/about_data (NASA photo of the day)
2. https://newsapi.org/docs/endpoints/top-headlines (News article)
